# 15-helloGH-jwiener
## Mein erstes GitHub-Projekt

Dieses Projekt dient dazu, die Grundlagen der Versionskontrolle mit Git und GitHub zu erlernen.

### Ziel

Das Hauptziel dieses Repositories ist es, den grundlegenden GitHub-Workflow zu verstehen und praktisch anzuwenden. Dazu gehören:

- Erstellen eines Repositories
- Arbeiten mit Commits
- Nutzung von Branches
- Push und Pull von Änderungen
- Verständnis von Versionierung und Änderungsverlauf

### Inhalt

Dieses Repository enthält einfache Übungsdateien, die schrittweise bearbeitet und versioniert werden.

### Lernziel

Am Ende dieses Projekts soll ein sicherer Umgang mit GitHub für grundlegende Entwicklungsprozesse erreicht werden.

Änderung auf lokaler Kopie.
